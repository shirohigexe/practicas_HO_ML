# practicas_HO_ML

Repository for practical work based on *Hands-On Machine Learning with Scikit-Learn, Keras, and PyTorch* by Aurélien Géron.

## Scope

The notebooks in this repository cover topics such as:

- data preparation and feature engineering
- supervised/unsupervised learning models
- model evaluation and validation
- hyperparameter tuning
- selected deep learning exercises with PyTorch

## Notes

- This is a study-oriented repository.
- Content is updated incrementally as new chapters and experiments are completed.
